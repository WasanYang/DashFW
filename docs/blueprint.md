# **App Name**: DevFlow Pro

## Core Features:

- Dynamic Kanban Board: Visualize and manage project statuses with customizable columns (e.g., Backlog, In Progress, Review, Completed, Paid), including drag-and-drop functionality for task cards.
- Smart Deadline & Revision Tracker: Each task card on the Kanban board automatically displays a real-time deadline countdown and tracks the number of client revision requests.
- AI-Powered Snippet Tool: Generate and manage quick-copy text snippets for common client communications, with an AI tool to suggest context-aware replies or relevant information for outbound messages.
- Integrated Client & Asset CRM: Store and manage client contact details, Fastwork profile links, and dedicated property modules for OTA jobs, including encrypted IDs/logins and links to Google Drive asset folders.
- Automated Financial Dashboard: Input gross project prices to automatically calculate net profit after Fastwork fees and view a summarized dashboard of 'Money in Pipeline' versus 'Withdrawn Earnings.'
- Customizable Checklists & Templates: Create and manage pre-defined, reusable checklists for various service types (e.g., 'New OTA Listing Setup' or 'Google Business SEO') to standardize workflow processes.

## Style Guidelines:

- The palette aims for a clean, professional, and tech-forward aesthetic. Primary color: A rich, professional blue (#1466B8) that evokes efficiency and clarity. It is well-suited for a modern developer and marketer's tool. Background color: A subtle, desaturated light blue (#F0F2F5) providing a calming yet professional canvas. Accent color: A vibrant cyan (#3CDDDD) to highlight interactive elements and draw attention to key actions, creating dynamic contrast without being distracting.
- Body and headline font: 'Inter' (sans-serif) for its highly legible, modern, and objective appearance, suitable for a data-dense dashboard. Code font: 'Source Code Pro' (monospace sans-serif) for any displayed code snippets, ensuring clear readability.
- Use clean, minimalistic, and vector-based icons, consistent with the Shadcn/UI aesthetic. Icons should intuitively represent their functions and contribute to quick information scanning on the dashboard.
- A responsive, modular dashboard layout with clear divisions for Kanban, CRM, and financial data. Emphasize efficient information density, easy navigation, and customization of panel arrangements suitable for a developer's workflow.
- Incorporate subtle, functional animations such as smooth transitions for Kanban card movements, visual feedback on form submissions, and progress indicators, enhancing the user experience without introducing distractions.
